   ```bash
   npm install
   ```

2. **Environment Variables**:

   ```env
   VITE_API_BASE_URL=http://localhost:8000
   ```
3. **Run in development mode**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```
